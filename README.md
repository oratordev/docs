# Orator Documentation

Documentation for [Orator](https://orator.dev) — the orator for your codebase.

Orator is a universal codebase context engine that indexes codebases and serves structured context to any AI coding tool via API and integrations.

## Development

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

## Deploy

Deployed automatically to [docs.orator.dev](https://docs.orator.dev) via Cloudflare Pages on push to `main`.
