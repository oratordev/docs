## What It Will Do

The Orator GitHub App will connect directly to your GitHub repositories for seamless integration:

- **Automatic indexing** — Re-index on every push to your default branch, so context is always up to date
- **PR context** — Automatically add relevant codebase context as PR comments, helping reviewers understand the impact of changes
- **Webhook-driven** — No polling or manual triggers needed; Orator reacts to GitHub events in real time
- **Team onboarding** — New team members can get codebase context from day one without any local setup
