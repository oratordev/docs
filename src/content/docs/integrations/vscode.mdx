## What It Will Do

The Orator VS Code extension will bring codebase context directly into your editor:

- **Inline context** — Hover over functions and classes to see their relationships, callers, and usage patterns from the knowledge graph
- **Context panel** — A dedicated sidebar showing relevant context for the file you're currently editing
- **Command palette integration** — Query Orator directly from VS Code with `Orator: Query Context`
- **Status bar** — See your repository's indexing status at a glance
- **Auto-index on save** — Optionally trigger incremental re-indexing when you save files
